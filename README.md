# 🎧 Music Listening Digital Twin

This project is a simple Digital Twin of a user's music listening habits.

## 🚀 Features
- Track songs listened
- Assign mood (happy, sad, chill, etc.)
- View listening statistics
- Visualize trends

## 🧠 What is a Digital Twin?
A Digital Twin is a virtual representation of a real-world system.  
This project models your music behavior as a digital system.

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Python (RAG simulation)
- JSON (data storage)

## ▶️ How to Run

1. Open `index.html` in browser
2. Add songs and moods
3. View stats update live
